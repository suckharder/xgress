# xgress - a proxy manager for Traefik

xgress is, in short, a **"Traefik Proxy Manager"**: what **Nginx Proxy Manager (NPM)**
is for nginx, xgress is for **Traefik v3**. Manage reverse-proxy hosts, TLS certificates,
and middleware from a clean admin UI - no hand-written labels or YAML.

> **xgress is an independent project. It is not affiliated with, endorsed by, or
> sponsored by Traefik Labs.** "Traefik" is a trademark of Traefik Labs; it is used
> here only to describe what xgress works with.
